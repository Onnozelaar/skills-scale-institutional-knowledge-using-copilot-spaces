# Data & Metrics Checklist

Checklist for instrumentation and measurement before release.

- [ ] Success metrics defined and documented in the one-pager or ticket
- [ ] Events and metrics instrumented for critical user flows
- [ ] Dashboards created or updated to surface key metrics
- [ ] Alerting thresholds defined for regressions in key metrics
- [ ] Ownership for measurement and analysis assigned
- [ ] Post-release analysis plan drafted
