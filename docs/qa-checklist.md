# QA & Testing Checklist

Checklist to ensure quality gates are met before a release.

- [ ] Unit tests added/updated and passing
- [ ] Integration tests added/updated and passing
- [ ] End-to-end smoke tests executed in staging
- [ ] Regression tests for critical flows identified and run
- [ ] Test data and environment validated
- [ ] Manual acceptance criteria verified (if applicable)
- [ ] QA status and outstanding issues shared in release notes
